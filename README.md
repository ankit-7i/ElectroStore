<!-- ══ GETTING STARTED ══ -->
<div align="center">

<img src="https://capsule-render.vercel.app/api?type=rect&height=48&text=%F0%9F%94%A7+GETTING+STARTED&fontSize=18&color=0:0d2137,100:050d1a&fontColor=00d4ff&animation=fadeIn" width="100%"/>

</div>

<br/>

### Prerequisites

```bash
# Required
Java 17+
Apache Tomcat 10.1
Maven 3.8+
MySQL / PostgreSQL
Docker (optional)
```

### Local Setup

```bash
# 1. Clone the repository
git clone https://github.com/ankit-7i/ElectroStore.git
cd ElectroStore

# 2. Configure database connection
# Edit src/main/java/com/project/util/DBConnection.java
# Update: DB_URL, USERNAME, PASSWORD

# 3. Build the project
mvn clean package

# 4. Deploy to Tomcat
cp target/ElectroStore.war $TOMCAT_HOME/webapps/

# 5. Start Tomcat
$TOMCAT_HOME/bin/startup.sh

# 6. Access the app
open http://localhost:8080/ElectroStore
```

### Docker Setup

```bash
# Build Docker image
docker build -t electrostore .

# Run container
docker run -p 8080:8080 electrostore

# Access
open http://localhost:8080
```
